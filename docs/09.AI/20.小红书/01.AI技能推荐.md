## 💡 一个观点

**AI Skill 的竞争拼的不是技术，是分发效率。** 谁能让不会写代码的人最快用上 AI，谁就赢。小红书用"种草"逻辑做 AI 工具分发，触达效率比 GitHub 高一个量级。这些 Skill 大多以小红书帖子为首要入口，GitHub 只是备份，这是属于这个时代的发行方式。

---

## 🎨 图文卡片 & 封面

### 🛠️ 归藏 Social Card Skill

28 种版式 + 10 套主题，AI 直接生成小红书图文卡和公众号封面。3400+ 人已经在用，搜"小红书 AI 作图教程"底下大概率挂着它。

- 覆盖小红书卡片、公众号 2:1 封面、1:1 方形封面等多种比例
- 内置主题系统，一键切换风格，不用设计基础也能出图

🔗 GitHub：<https://github.com/op7418/guizang-social-card-skill>

---

### ⭐ 宝玉 baoyu-skills

16K+ GitHub Star，自媒体人专属视觉工具箱。图文卡片、信息图、封面、配图，一套全搞定。这是目前 GitHub 上 Star 数最高的中文自媒体 AI Skill 仓库之一。

- 涵盖从选题到配图的完整视觉生产链路
- 适合需要批量产出图文内容的创作者

🔗 GitHub：<https://github.com/JimLiu/baoyu-skills>

## ✍️ 爆款写作

### 小红书爆款笔记生成 Skill

基于 gpt-image-2 的爆款笔记生成器。图生图卡片 + 5 种标题公式——

1. **利诱型**：直击利益点
2. **反常识型**：打破认知预期
3. **情感共鸣型**：触发情绪共振
4. 另外两种覆盖更多场景

支持自然语言修改，一句话调整内容方向。

🔗 GitHub：<https://github.com/JuneYaooo/xhs-writer-skill>

## 📊 账号运营

### 小红书运营 Skill

从内容生产升级到完整账号运营——推荐流分析、自动发布、爆款复刻、评论回复，一个 Skill 覆盖日常运营全流程。

有用户搭建了"一人三号"方案，14 天跑通矩阵冷启动。适合需要多账号、高频更新的自媒体团队。

🔗 GitHub：<https://github.com/Xiangyu-CAS/xiaohongshu-ops-skill>

## 📝 内容质量

### AI Content Skills（反 AI 味词库）

核心差异化：**内置反 AI 味词库**，专门解决"读起来像 AI 写的"这个自媒体创作者最头疼的问题。用 AI 生产内容已经不难，难的是让读者看不出来是 AI 写的。

- 自动检测 AI 痕迹词句
- 提供符合中文语感的替代方案
- 适配小红书、公众号、知乎等多平台语态

> ⚠️ 暂未找到公开发布的 GitHub 仓库，可能通过小红书帖子以 Skill 文件或 Gist 形式分发。搜小红书"反AI味词库"可找到最新版。

## 🚀 多平台分发

### 陈斤 chenjin-cmd

微信 + 小红书 + 抖音 + Twitter 四平台起号方案，一个 Skill 覆盖全栈分发。内容一次生产，适配多平台格式自动发布，适合跨平台矩阵运营。

> ⚠️ 暂未找到公开的 GitHub 仓库。这类全平台分发 Skill 通常以小红书帖子 + Skill 文件形式发布，建议小红书搜"chenjin-cmd"获取安装方式。

🔗 GitHub：<https://github.com/oil-oil/oil-cover>

## ⚙️ 工作流搭建

### Skill Workflow

五步流水线：**爬取 → 清洗 → 分析 → 创作 → 格式化**。基于 Claude Agent SDK，零经验半小时搭一套内容生产系统。

适合有长期内容产出需求、想把重复性工作自动化的团队。

> ⚠️ 暂未找到同名公开仓库。这类工作流项目多通过 ClawHub / COC 市场或小红书帖子以 Skill 文件形式分发，可搜"skill-workflow Claude Agent SDK"。
