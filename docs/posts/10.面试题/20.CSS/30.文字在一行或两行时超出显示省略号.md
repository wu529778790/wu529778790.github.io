## 一行超出显示省略

```css
overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;
```

<!-- more -->

```html
  </div>

  .box-42b6 {
    border: 1px solid #999;
    width: 200px;

    overflow: hidden;

</style>
```

## 两行（多行）超出显示省略号

```css
overflow: hidden;
white-space: normal;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 2;
-webkit-box-orient: vertical;
```

显示的行数由`line-clamp`样式的值决定。

:::demo [vanilla]

```html
<html>
  <div class="box2-42b6">
    演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字演示文字
  </div>
</html>
<style>
  .box2-42b6 {
    border: 1px solid #999;
    width: 200px;

    overflow: hidden;
    white-space: normal;
    text-overflow: ellipsis;
    display: -webkit-box;
    -webkit-line-clamp: 2;
    -webkit-box-orient: vertical;
  }
</style>
```
