# AI Agent 开发要学什么

大模型很强，但它有几个硬伤：

- **没记忆** — 昨天聊的今天全忘了
- **没手** — 只能告诉你怎么做，不能帮你做
- **没私有知识** — 公司内部文档它一概不知
- **没时事** — 训练数据截止之后的事它不知道

所以 Agent 要做的，就是给大模型打补丁：

- **Memory** — 让它记住该记的，忘记该忘的
- **Tool** — 让它能读写文件、跑命令、开浏览器，真正动手
- **RAG** — 接入私有知识库，查到它本来不知道的东西

大模型本身就具备思考和规划能力，加上这三样，就是一个能记住你、能帮你干活、能查内部知识的 Agent。

平时用的 Cursor、Manus，本质上都是这个思路 — 用 Tool 把能力扩展到模型之外。

接下来要学的东西：RAG、Memory、Tool、MCP、Multi Agent，以及开发框架 **LangChain**（Agent 框架）和 **LangGraph**（多 Agent 框架）。
